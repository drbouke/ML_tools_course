Deploy the Saved Heart Disease Model as a Flask API

Topics: Model Serialization, Flask API, JSON Input Validation, Prediction Endpoint, Model Deployment

Exercise Context

In the previous exercise, you cleaned the Heart Disease dataset, built a preprocessing pipeline, trained a classification model, and evaluated its performance.

In this new exercise, you will use the same trained model after saving it to disk, then create a Flask API that allows external applications to send patient data and receive predictions.

Goal

The goal is to deploy the saved heart disease prediction pipeline as a web API using Flask.

Your API should:

  • Load the saved trained pipeline at application startup
  • Expose a health check endpoint
  • Expose a schema endpoint that documents the expected input columns
  • Expose a prediction endpoint that accepts JSON data
  • Return model predictions in JSON format

Step 3: Create the Flask Application

Create a Flask app that loads the saved model once when the server starts. 


from pathlib import Path
import joblib
import pandas as pd
from flask import Flask, request, jsonify

app = Flask(__name__)

BASE_DIR = Path(__file__).resolve().parent
MODEL_PATH = BASE_DIR / "heart_pipeline.joblib"

EXPECTED_COLUMNS = [
    "Age",
    "Sex",
    "ChestPain",
    "RestBP",
    "Chol",
    "Fbs",
    "RestECG",
    "MaxHR",
    "ExAng",
    "Oldpeak",
    "Slope",
    "Ca",
    "Thal"
]

if not MODEL_PATH.exists():
    raise FileNotFoundError(f"Saved model not found: {MODEL_PATH}")

model = joblib.load(MODEL_PATH)

Step 8: Add the Prediction Endpoint

The API should validate the JSON body, verify required columns, then run the saved model. 


@app.post("/predict")
def predict():
    payload = request.get_json(silent=True)

    if payload is None:
        return jsonify({"error": "Request body must be valid JSON."}), 400

    if "records" not in payload:
        return jsonify({"error": "JSON body must contain a 'records' field."}), 400

    records = payload["records"]

    if not isinstance(records, list) or len(records) == 0:
        return jsonify({"error": "'records' must be a non-empty list."}), 400

    if not all(isinstance(record, dict) for record in records):
        return jsonify({"error": "Each item in 'records' must be a JSON object."}), 400

    try:
        X_new = pd.DataFrame(records)

        missing_columns = [col for col in EXPECTED_COLUMNS if col not in X_new.columns]
        extra_columns = [col for col in X_new.columns if col not in EXPECTED_COLUMNS]

        if missing_columns:
            return jsonify({
                "error": "Missing required columns.",
                "missing_columns": missing_columns,
                "extra_columns": extra_columns
            }), 400

        X_new = X_new[EXPECTED_COLUMNS]

        predictions = model.predict(X_new).tolist()

        response = {
            "predictions": predictions,
            "n_records": len(records)
        }

        if hasattr(model, "predict_proba"):
            response["predict_proba"] = model.predict_proba(X_new).tolist()

        return jsonify(response), 200

    except Exception as e:
        return jsonify({"error": str(e)}), 500
